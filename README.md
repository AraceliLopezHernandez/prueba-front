# prueba-front

Proyecto frontend base con una pagina HTML inicial que utiliza Bootstrap 5 por CDN.

## Contenido

- `index.html`: estructura inicial del proyecto con el mensaje "Hola mundo".

## Tecnologias

- HTML5
- Bootstrap 5

## Como usar

1. Clona este repositorio.
2. Abre `index.html` en tu navegador.

## Estructura

```text
prueba-front/
|-- index.html
|-- README.md
`-- .gitignore
```
